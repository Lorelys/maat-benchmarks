# Controlled E-commerce Agency Benchmark

15-agent multi-marketplace e-commerce agency workflow.

Grounded in Uliana Elina's professional experience at a mid-market agency (70+ clients, 60+ employees).

Failure profiles correspond to defect categories observed in production operations.

Scoring is fully deterministic — no LLM in the scoring path.

## Grid

8 failure profiles × 3 arms (`v3_off`, `v3_warn`, `v3_intervene`) × 5 seeds = **120 trials**.

Full measured results: `RESULTS_SUMMARY.md` in this folder and `results/controlled_ecommerce_agency/`.
